**How to Run**

Step 1 - Clone the repo

Step 2 - After cloning run pnpm i

Step 3 - To run the project - pnpm run dev


**Technologies**

React + Typescript, pnpm, Python (Backend), Flowbite, Lucid React, Tailwind CSS, Zod, Zustand.
